## 🆘 TROUBLESHOOTING

### If deployment fails:

**1. Check Firebase login:**
```powershell
firebase logout
firebase login
```

**2. Verify project:**
```powershell
firebase use inventory-f8f66
```

**3. Try again:**
```powershell
firebase deploy
```

### If build fails:

```powershell
cd frontend
rm -rf node_modules
npm install
npm run build
```
